You can copy-paste the following script into a terminal window to get all the apps in this pack. Alternatively, you can download the .bat or .ps1 file to quickly install this pack.

BAT:

```
winget install --id=Microsoft.VisualStudio.Enterprise -e && winget install --id=Microsoft.VisualStudioCode -e && winget install --id=Microsoft.VisualStudio.BuildTools -e && winget install --id=Microsoft.PowerToys -e && winget install --id=AnyDeskSoftwareGmbH.AnyDesk -e && winget install --id=Audacity.Audacity -e && winget install --id=Klocman.BulkCrapUninstaller -e && winget install --id=FastCopy.FastCopy -e && winget install --id=Docker.DockerDesktop -e && winget install --id=dnGrep.dnGrep -e && winget install --id=SoftDeluxe.FreeDownloadManager -e && winget install --id=Git.Git -e && winget install --id=Axosoft.GitKraken -e && winget install --id=Opera.OperaGX -e && winget install --id=Mega.MEGASync -e && winget install --id=Notepad++.Notepad++ -e && winget install --id=Microsoft.SQLServerManagementStudio -e && winget install --id=OpenJS.NodeJS -e && winget install --id=Python.Python -e && winget install --id=Yarn.Yarn -e && winget install --id=Spotify.Spotify -e && winget install --id=VideoLAN.VLC -e && winget install --id=RARLab.WinRAR -e && winget install --id=Telegram.TelegramDesktop -e && winget install --id=Rambox.RamboxCE -e && winget install --id=WhatsApp.WhatsApp -e && winget install --id=SlackTechnologies.Slack -e
```
